# <a name="side-by-side-comparison-of-git-and-team-explorer"></a>Porównanie Side-by-Side dla usługi git i Team Explorer

Uruchomiono pierwszą wersję nowego środowiska Git w programie Visual Studio 2019 w [wersji 16,8](/visualstudio/releases/2019/release-notes/). To nowe środowisko pomaga ograniczyć przełączanie kontekstu przy użyciu prostego okna **zmiany git** , które obejmuje typowe zadania git. Obejmuje on również okno **repozytorium git** na całym ekranie dla bardziej zaawansowanych operacji git, takich jak zarządzanie gałęziami i przeglądanie repozytorium.

Jeśli używasz Team Explorer, poniżej przedstawiono przewodnik krok po kroku, w którym wyjaśniono, jak można użyć nowego środowiska git.

> [!NOTE]
> Poniższa sekcja zawiera rozmiar zrzutów ekranu do dopasowania do kolumn tabeli. Kliknij każdy zrzut ekranu, aby wyświetlić jego większą wersję. (Jeśli używasz urządzenia dotykowego, naciśnij każdy zrzut ekranu, aby wyświetlić jego większą wersję).
