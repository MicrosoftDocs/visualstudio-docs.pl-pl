# <a name="c6997"></a>C6997
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie 6997: adnotacje w tej lokalizacji są bezsensowne i zostaną zignorowane.  
  
 Nie można zastosować adnotacji `extern "C" {...}`. Zastosuj adnotacje do określonego obiektu.
