# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28226: Adnotacja także nie może być PrimOp (w bieżącej deklaracji)  
  
 To ostrzeżenie wskazuje, że jest on zadeklarowany jako PrimOp adnotacja o nazwie i również został wcześniej zadeklarowany jako normalny adnotacji. To nie powinna być może wystąpić, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w czasie normalnego użycia.
