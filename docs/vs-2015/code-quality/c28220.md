# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28220: oczekiwano wyrażenia typu Integer dla adnotacji, parametr w adnotacji  
  
 To ostrzeżenie wskazuje, że parametr do adnotacji powinien być wyrażeniem liczby całkowitej i został napotkany inny typ. Zwykle oznacza to nieprawidłowe kodowane makro adnotacji.
