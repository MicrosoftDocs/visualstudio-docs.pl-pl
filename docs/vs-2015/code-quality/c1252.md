# <a name="c1252"></a>C1252
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C1252: cykliczna lub brakująca zależność między wtyczkami: wymaga identyfikatora GUID  
  
 Narzędzie do analizy kodu raporty tego ostrzeżenia, gdy występuje błąd wewnętrzny z zależnościami wtyczki, nie w kodzie, analizowane.
