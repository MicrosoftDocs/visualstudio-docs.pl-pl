# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28212: Oczekiwano wyrażenia wskaźnikowego dla adnotacji  
  
 To ostrzeżenie wskazuje, że numerowane parametrów do adnotacji (nie funkcja adnotacją) powinien być wskaźnika lub typ tablicy, ale napotkano innego typu. Adnotacja musi zostać poprawione.
