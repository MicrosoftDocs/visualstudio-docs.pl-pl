# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28216: \_Check_return\_ adnotacja ma zastosowanie tylko do warunków dla określonego parametru funkcji.  
  
 `_Check_return_` Adnotacji była stosowana w kontekście innej niż `__post`; może być konieczne `__post` (lub `__drv_out`) i/lub jej może znajdować się.
