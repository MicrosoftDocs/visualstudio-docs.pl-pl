# <a name="getting-started-with-source-control-vspackages"></a>Wprowadzenie do pakietów VSPackage kontroli kodu źródłowego
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Pakietu VSPackage kontroli źródła jest bardziej zaawansowanych alternatywa wtyczka do kontroli źródła. (Zobacz [tworzenie wtyczki kontroli źródła](../../extensibility/internals/creating-a-source-control-plug-in.md) więcej informacji na temat źródła kontrolowanie dodatków plug-in.) Pakietu VSPackage kontroli źródła umożliwia pełną kontrolę nad model kontroli źródła, funkcje i interfejs użytkownika (UI), a jej zintegrowania [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] środowiska jako pakietu VSPackage.

 Opisano opcje dla rozwiązania kontroli źródła oraz szerokiego wskazówki dotyczące wybierania ścieżką odpowiednie integracji.
