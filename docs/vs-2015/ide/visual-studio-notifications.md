# <a name="visual-studio-notifications"></a>Powiadomienia programu Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio posiada systemu powiadomień, które ostrzega o różnych zdarzeń:

1. Aktualizacja jest dostępna dla składnika w programie Visual Studio

2. Visual Studio jest dostępna aktualizacja

3. Było zdarzenia środowiska, które mogą być zainteresowane

   Visual Studio powiadamia, gdy aktualizacja jest dostępna dla programu Visual Studio lub któregokolwiek ze składników, które zostały zainstalowane. Umożliwia także powiadomienia dotyczące niektórych rodzajów zdarzeń, które miały miejsce w środowisku Visual Studio. Ikona powiadomień jest flaga w prawym górnym rogu pasek tytułu programu Visual Studio. Gdy ikona jest czerwony, oznacza to, że występuje problem z licencją. Jeśli jest żółty Brak dostępnej aktualizacji produktu Visual Studio do zainstalowania. Gdy jest to "normal" lub czarne zostaną wyświetlone komunikaty informacyjne lub opcjonalne, takie jak awarie, dla których możesz chcieć znaleźć obejście tego problemu. Możesz kliknąć ikonę aby otworzyć **Centrum powiadomień** i wybrać kolumny do działania w. Możesz przeprowadzić jednorazowe odrzucić na powiadomienie lub zawsze Ignoruj powiadomienia (przez umieszczenie nad nim kursora nad nim), i programu Visual Studio nigdy nie pokażą tego komunikatu ponownie.

   Jeśli zdecydujesz się zawsze Ignoruj powiadomienie, Visual Studio przestanie pokazywania go. Możesz zresetować listy ignorowanych powiadomień w [powiadomienia, środowisko, opcje, okno dialogowe](../ide/reference/notifications-environment-options-dialog-box.md).
