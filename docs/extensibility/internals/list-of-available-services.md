# <a name="list-of-available-services"></a>Lista dostępnych usług

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] zestaw SDK programu Visual Studio obsługuje następujące usługi. Niektóre pakiety oferują własne usługi, które nie są wymienione w tym miejscu — na przykład usługi językowe nie mają pojedynczego identyfikatora GUID usługi. Należy użyć nazwy języka, aby znaleźć identyfikator GUID usługi językowej w rejestrze.

Użyj identyfikatorów GUID usługi wymienionych tutaj lub uzyskanych z innych źródeł (na przykład usług językowych), aby uzyskać podstawowy interfejs lub interfejsy wyświetlane dla każdej usługi.
