# <a name="list-of-available-services"></a>Lista dostępnych usług

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] Zestaw SDK Visual Studio obsługuje następujące usługi. Niektóre pakiety oferują własne usługi, których nie wymieniono tutaj — na przykład usługi językowe nie mają jednego identyfikatora GUID usługi. Należy użyć nazwy języka, aby znaleźć identyfikator GUID usługi językowej w rejestrze.

Użyj identyfikatorów GUID usługi wymienionych w tym miejscu lub uzyskanych z innego źródła (na przykład usług językowych), aby uzyskać podstawowy interfejs lub interfejsy wyświetlane dla każdej usługi.
