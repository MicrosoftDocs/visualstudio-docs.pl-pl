# <a name="c28020"></a>C28020
Ostrzeżenie C28020: wyrażenie \<expression > nie jest prawdziwe w tym wywołaniu

 To ostrzeżenie jest zgłaszane, gdy \_jest zgodne z wyrażeniem\_ na liście nie ma wartości true. Często wskazuje to na nieprawidłowy parametr.

 Jeśli wystąpi to w deklaracji funkcji, adnotacje wskazują na niemożliwy warunek.
