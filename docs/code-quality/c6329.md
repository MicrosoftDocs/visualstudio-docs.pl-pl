# <a name="c6329"></a>C6329
Ostrzeżenie C6329: Wartość zwracana dla wywołania \<function > nie powinna być sprawdzana względem \<number >

 Program porówna liczbę z wartością zwracaną z wywołania do `CreateFile`. Jeśli `CreateFile` powiedzie się, zwraca otwarte dojście do obiektu. Jeśli to się nie powiedzie, zwraca `INVALID_HANDLE_VALUE`.

## <a name="example"></a>Przykład
 Ten kod może spowodować Ostrzeżenie:

```cpp
if (CreateFile() == NULL)
    {
        return;
    }
```

## <a name="example"></a>Przykład
 Ten kod poprawia błąd:

```cpp
if (CreateFile() == INVALID_HANDLE_VALUE)
    {
        return;
    }
```
