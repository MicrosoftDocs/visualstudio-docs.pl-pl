# <a name="side-by-side-comparison-of-git-and-team-explorer"></a>Porównanie obok siebie usługi Git i Team Explorer

W wersji [16.8](/visualstudio/releases/2019/release-notes/)uruchomiliśmy pierwszą wersję nowego Visual Studio Git. To nowe środowisko ułatwia ograniczenie przełączania kontekstu za pomocą prostego okna **zmian Usługi Git,** które zawiera typowe zadania usługi Git. Zawiera również okno repozytorium **Git** na całej ekranie, które oferuje bardziej zaawansowane operacje usługi Git, takie jak zarządzanie gałęziami i przeglądanie repozytoriów.

Jeśli korzystasz z usługi Team Explorer, oto przewodnik krok po kroku, w których wyjaśniono, jak korzystać z nowego sposobu korzystania z usługi Git.

> [!NOTE]
> W poniższej sekcji znajdują się zrzuty ekranu dopasowane do kolumn tabeli. Kliknij każdy zrzut ekranu, aby wyświetlić jego większą wersję. (Jeśli używasz urządzenia dotykowego, naciśnij każdy zrzut ekranu, aby wyświetlić jego większą wersję).
