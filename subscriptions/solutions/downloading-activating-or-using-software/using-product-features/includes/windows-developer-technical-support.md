## <a name="windows-developer-technical-support"></a>Pomoc techniczna dla deweloperów systemu Windows  

Obsługa subskrypcji programu Visual Studio zapewnia pomoc dotyczącą sprzedaży, subskrypcji, kont i rozliczeń za subskrypcje programu Visual Studio. Jeśli masz problem z kontem technicznym lub deweloperem systemu Windows, Twoje pytanie musi zostać skierowane do [pomocy technicznej dla deweloperów systemu Windows](https://developer.microsoft.com/windows/support/?utm_source=developermscom). Zalecamy, aby najpierw próbować wykonać te inne zasoby. 

## <a name="windows-technical-documentation"></a>Dokumentacja techniczna systemu Windows 
Przejrzyj samouczki, przewodniki dotyczące rozwiązywania problemów, przydatne zasoby i różne tematy w [dokumentacji technicznej systemu Windows](https://docs.microsoft.com/windows/).

## <a name="windows-developer-account"></a>Konto dewelopera systemu Windows
Twoja subskrypcja programu Visual Studio może obejmować konto dewelopera systemu Windows. Aktywuj swoje konto, odwiedzając [stronę korzyści portalu subskrypcji](https://my.visualstudio.com/Benefits) i klikając kafelek korzyści. 

Przeczytaj więcej na temat [korzyści dla deweloperów systemu Windows](https://docs.microsoft.com/visualstudio/subscriptions/vs-windows-dev).
